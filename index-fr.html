Questions fréquentes

Comment restaurer mon achat Premium ?

Ouvrez Pecunia et utilisez l’option Restaurer les achats. Vérifiez que vous êtes connecté avec le même compte Apple que celui utilisé lors de l’achat initial.

Où mes données sont-elles stockées ?

Vos transactions, catégories, budgets et autres informations financières sont stockés localement sur votre appareil et synchronisés via iCloud. Le développeur ne peut pas accéder à vos données personnelles ou financières.

Comment récupérer mes données sur un nouvel iPhone ?

Connectez-vous au nouvel appareil avec le même compte Apple et assurez-vous qu’iCloud est activé pour Pecunia. Vos données devraient se synchroniser automatiquement.

Pourquoi mes données ne se synchronisent-elles pas ?

  • Vous êtes connecté à iCloud.
  • iCloud est activé pour Pecunia dans les réglages.
  • Vous utilisez le même compte Apple sur tous les appareils.
  • Votre appareil dispose d’une connexion Internet active.
  • Vous disposez de suffisamment d’espace iCloud.

Comment supprimer mes données ?

Vous pouvez supprimer les données locales en désinstallant Pecunia. La désinstallation peut ne pas supprimer automatiquement les copies synchronisées avec iCloud ; pour les supprimer, gérez les données Pecunia dans les réglages de votre compte Apple.

Pecunia collecte-t-elle des données personnelles ?

Non. Pecunia ne nécessite aucun compte et n’utilise aucun outil d’analyse, de publicité ou de suivi.

Comment fonctionne l’achat Premium ?

Pecunia propose un achat intégré unique qui débloque toutes les fonctionnalités Premium. Apple traite le paiement de manière sécurisée via l’App Store.

Signaler un problème

Lors du signalement d’un problème, indiquez autant d’informations que possible :

  • Modèle de votre iPhone ou iPad.
  • Version d’iOS ou d’iPadOS.
  • Version de Pecunia.
  • Description claire du problème.
  • Étapes permettant de reproduire le problème.
  • Captures d’écran, si elles ne contiennent pas d’informations sensibles.

N’incluez pas d’informations financières, de détails de transactions, de mots de passe ou d’autres données sensibles dans les e-mails d’assistance.
